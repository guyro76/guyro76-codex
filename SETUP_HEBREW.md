# 🚀 LangFlip Desktop — התחלה מהירה (Windows)

**בצע את הצעדים הבאים ובעוד 10 דקות ההאפליקציה תרוץ.**

---

## 📥 **שלב 1: הורדת .NET 8.0** (אם עדיין אין)

1. גלוש ל: https://dotnet.microsoft.com/download
2. בחר **.NET 8.0** (LTS)
3. בחר **Windows** → **x64**
4. הורד את ה-installer
5. הרץ את ה-installer (ליחצו Next, Next, Install)
6. **השלם את ההתקנה** (אולי צריך הפעלה מחדש)

**בדוק התקנה:**
```
Windows Key + R
cmd
dotnet --version
```

צריך להראות: `8.0.xxx`

---

## 💾 **שלב 2: השתנסח את הקוד**

1. גלוש לכאן: https://github.com/guyro76/guyro76-codex
2. לחץ **Code** (ירוק, משמאל)
3. בחר **Download ZIP**
4. חוקם את ה-ZIP למשל ל: `C:\LangFlip`

---

## 🔨 **שלב 3: בנייה**

### **דרך קלה (עבור Windows):**

1. פתח את התיקייה שהורדת
2. פעול **כפול-לחץ** על `build.bat`
3. חכה שיסיים (צריך ~30 שניות)
4. אם הכל תקין - תראה: `Build successful ✅`

### **דרך עם Command Prompt:**

```
Windows Key + R
cmd
cd C:\LangFlip
build.bat
```

---

## ▶️ **שלב 4: הרצה**

אחרי שהבנייה הסתיימה בהצלחה:

```
cd C:\LangFlip
dotnet run -c Release
```

**או:**

1. פתח את התיקייה: `C:\LangFlip`
2. בתוך תיקייה: `bin\Release\net8.0-windows10.0.19041.0\`
3. כפול-לחץ על: **LangFlipDesktop.exe** ✅

---

## 🧪 **שלב 5: בדיקה**

1. האפליקציה נפתחת ✓
2. פתח Notepad
3. הקלד: `akuo`
4. סמן את הטקסט (Ctrl+A)
5. לחץ: **Ctrl + Alt + H**
6. התוצאה צריכה להיות: **שלום** ✓

**אם זה עבד - כל דבר עובד!** 🎉

---

## 🔑 **שלב 6 (בחר): בדיקת תרגום (עם OpenAI)**

### **אם אתה רוצה תרגום:**

1. הולך ל: https://platform.openai.com/api-keys
2. **Sign Up** או **Sign In**
3. לחץ **Create new secret key**
4. העתק את ה-key (מתחיל עם `sk-`)
5. חזור ל-LangFlip
6. לחץ **הגדרות**
7. בחר **OpenAI**
8. הדבק את ה-API Key
9. לחץ **בדוק חיבור**

**אם הכל ירוק**, תרגום עובד! ✓
