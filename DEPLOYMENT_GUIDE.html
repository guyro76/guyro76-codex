🚀 Deploy ל-Vercel

צעדים:

שלב 1: בחר Repository
• לחץ "Continue with GitHub"
• חפש guyro76-codex
• לחץ Import

שלב 2: הוסף Environment Variables
• ראה את רשימת ה-Variables מטה
• הוסף כל אחד ב-Vercel

שלב 3: Deploy
• לחץ הכפתור האדום "Deploy"
• חכה 2-3 דקות

Environment Variables (להוסיף ב-Vercel):

NEXTAUTH_SECRET=c81da87e10a38cdd40a2a0d4351e7ec15d03a001827c61cdeaf760c8c80b7384 NEXTAUTH_URL=https://guyro76-codex.vercel.app DATABASE_URL=postgresql://postgres:[PASSWORD]@[PROJECT].supabase.co:5432/postgres CLAUDE_API_KEY=sk-ant-[YOUR_API_KEY]

⚠️ החלף:

  • [PASSWORD] = הסיסמה של Supabase
  • [PROJECT] = שם ה-Project של Supabase
  • [YOUR_API_KEY] = Claude API Key שלך
