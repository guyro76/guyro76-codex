# Paprika Campaign Inspector

תוסף דפדפן מקומי שמציג חלונית צד בתוך Google Ads ו-Meta Ads Manager.

## התקנה

1. פתח את מסך ניהול התוספים בדפדפן Chrome.
2. הפעל מצב מפתחים.
3. בחר טעינת תוסף לא ארוז.
4. בחר את התיקייה `paprika-app/browser-extension`.
5. פתח מחדש את מסך ניהול הקמפיינים ולחץ על אייקון פפריקה.

## שימוש

1. בחר במערכת הפרסום את טווח התאריכים ואת העמודות הרצויות.
2. פתח מסך Campaigns, Ads, Keywords או Search Terms.
3. בחלונית פפריקה לחץ על סריקת הטבלה הגלויה.
4. העתק את הטבלה כ-TSV והדבק אותה במסך בדיקת הקמפיינים בפפריקה, או הורד Snapshot כ-JSON.

## פרטיות

התוסף קורא רק כותרות וטבלאות שמוצגות כרגע במסך. הוא אינו קורא סיסמאות, שדות טופס, Cookies, Tokens או טקסט מלא מכל העמוד.

## עמודות מומלצות

Google Ads: Cost, Impressions, Clicks, Conversions, Conversion Value, CTR, CPC, Search Term, Keyword ו-ROAS.

Meta Ads: Amount Spent, Impressions, Link Clicks, Results, Purchase Conversion Value, CTR, CPC, CPM, Frequency ו-Purchase ROAS.
