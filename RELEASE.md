## ⭐ **מה זה קליקשפה?**

אפליקציה חכמה עבור Windows שתיקנה טעויות מקלדת, מתרגמת, ומשפרת כתיבה בעברית ובאנגלית.

- ✅ **תיקון טעויות מקלדת** (Hebrew ↔ English)
- ✅ **תרגום מיידי** (עם OpenAI)
- ✅ **שיפור כתיבה** (Hebrew & English)
- ✅ **עובד בכל מקום** (Hotkeys)
- ✅ **ממשק יפה** (Glassmorphism)
- ✅ **הגנת פרטיות** (No tracking)

## 🌟 **Features:**

### **Keyboard Correction (עבודה מהלא)**
- English QWERTY → Hebrew: `akuo` → `שלום`
- Hebrew → English: `שךםצ` → `shop`
- No internet needed, instant

### **Translation (עם OpenAI)**
- Hebrew → English
- English → Hebrew
- Requires API key

### **Writing Improvement**
- Make text clearer
- More professional
- Better flow

### **Access Methods**
- Global hotkeys (anywhere)
- System tray (right-click)
- Floating menu (Ctrl+Alt+Space)
- Main dashboard (buttons)
